#系统说明
1、本系统后端中使用到mssql、mybatis 、springboot、security、redis、JWt，第一次系统运行请在api.java 阅读apiNode进行设置超级管理员账户

2、前端使用Vue2.0 (注：3.0当时我并不知情已经发布及element并未更新支持到3.0)及element作为页面美化

3、超级管理员账户下可设置教师账户

4、教师有添加学生，添加课表，添加学生成绩，将学生添加入自己的课程内等

5、学生有查询成绩
